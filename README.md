# Evolvix Global Group

**The Architecture of Success.**

Evolvix Global is a conceptual digital experience representing an international holding company that leads the evolution of strategic assets and services. The project is designed under the aesthetic principles of *"Lujo Tecnológico"* (Technological Luxury) and *"Minimalismo Autoritario"* (Authoritative Minimalism), translating complex corporate structures into a highly refined, premium user interface.

## 🌐 The Ecosystem

Evolvix operates as an integrated system of four interdependent divisions, designed to sustain frictionless growth at an international scale:

1. **Strategic Logistics & Supply Chain:** Absolute synchronization. Turning logistical complexity into a global competitive advantage.
2. **Global Consulting & Business Intelligence:** Intelligence precedes action. Designing the roadmap for the expansion of leading groups.
3. **Asset Management & Investment:** Strength in management, vision in investment. High-precision financial architecture.
4. **Innovation & Digital Infrastructure:** Building tomorrow, today. Sovereign platforms, data fabric, and AI orchestration.

## 📍 Global Footprint

A coordinated network that synchronizes capital, market, and execution across four major global nodes:
- **Madrid:** Operations Hub, Group Strategy, EU Compliance
- **London:** Capital Markets, Investor Relations, Cross-Border Advisory
- **Dubai:** Emerging Logistics, MENA Trade, Sovereign Mandates
- **Singapore:** Innovation Division, APAC Engineering, AI Infrastructure

The footprint is visualized through an interactive, geographically accurate `react-simple-maps` projection using Natural Earth data.

## 🛠️ Technical Architecture

This project is built for high performance, utilizing a modern frontend stack:
- **Framework:** React 19 + TanStack Start (File-based routing)
- **Styling:** Tailwind CSS v4 with a custom semantic design system.
- **Typography:** Custom implementation of *Clash Display* (Headings) and *Inter* (Body/Mono labels).
- **Internationalization (i18n):** Custom React Context architecture providing seamless, zero-reload switching between English and Spanish across the entire platform.
- **Animations:** GSAP & Tailwind-based micro-interactions, custom reveal components, and magnetic cursors.
- **Map Engine:** `react-simple-maps` with `geoNaturalEarth1` projection for realistic geographical rendering.

## 🎨 Design System

The visual language is built upon a strict color palette and typographic hierarchy:
- **Background:** Absolute Black (`#050505`) for maximum contrast and depth.
- **Typography:** Titanium White (`#f7f7f7`) and Platinum (`#d1d1d1`) for high legibility, accented by Smoke Gray (`#9e9e9e`).
- **Accents:** Electric Blue (`#ad30fe`) used sparingly for interactive highlights and geographical markers.
- **Micro-aesthetics:** Glassmorphism (`backdrop-blur`), hairline borders, and subtle vignette gradients to create depth without clutter.
